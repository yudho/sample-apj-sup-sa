# Mistral Small 3.2 24B benchmark

Per-model configuration for [`mistralai/Mistral-Small-3.2-24B-Instruct-2506`](https://huggingface.co/mistralai/Mistral-Small-3.2-24B-Instruct-2506),
a 24B-parameter Apache-2.0 instruction-tuned model. This folder is the only
place Mistral-Small-3.2-specific facts live — everything else is generic
infrastructure in [`src/vllm_ec2_bench/`](../../src/vllm_ec2_bench/).

## Layout

```
models/mistral_small_3_2_24b/
├── __init__.py          # Re-exports MODEL spec, EXPERIMENTS, prompts, catalog factories
├── model_spec.py        # MISTRAL_SMALL_3_2_24B: ModelSpec
├── experiments.py       # EXPERIMENTS: 7 ExperimentConfig instances
├── prompts.py           # Travel-booking JSON-extraction SYSTEM_PROMPT + SEED_INPUT
├── catalog_cache.json   # Hardware + prices cache (schema v2, checked-in)
└── mistral-small-3-2-24b-vllm-ec2-benchmark.ipynb   # Generated notebook
```

## Experiments (Mistral Small 3.2 24B at BF16, ~55 GiB)

| # | Instance | GPUs | TP | DP | PP | Replicas | Notes |
|---|---|---|---|---|---|---|---|
| exp_1 | `g5.12xlarge`  | 4× A10G        | 4 | 1 | 1 | 1 | TP=4 across A10Gs (22 GiB ea) |
| exp_2 | `g6.12xlarge`  | 4× L4          | 4 | 1 | 1 | 1 | TP=4 across L4s (24 GiB ea) |
| exp_3 | `g6e.12xlarge` | 4× L40S        | 2 | 2 | 1 | 2 | 2 replicas, TP=2 each |
| exp_4 | `g7e.2xlarge`  | 1× Blackwell   | 1 | 1 | 1 | 1 | smallest fit; ~55 GiB on a single Blackwell |
| exp_5 | `g7e.12xlarge` | 2× Blackwell   | 1 | 2 | 1 | 2 | 1 replica per GPU |
| exp_6 | `p4d.24xlarge` | 8× A100-40GB   | 2 | 4 | 1 | 4 | 4 replicas, TP=2 each |
| exp_7 | `p4de.24xlarge`| 8× A100-80GB   | 1 | 8 | 1 | 8 | dense pack, 1/GPU |

Spot first, on-demand fallback. All experiments default to `us-west-2`.

## Task

Travel booking JSON extraction. The endpoint is benchmarked against records
generated by `sample-data/scripts/synthesize.py --domain travel`.

## Usage

```python
from vllm_ec2_bench import DeploymentRunner
from models.mistral_small_3_2_24b import EXPERIMENTS, SYSTEM_PROMPT, load_catalog

CATALOG = load_catalog(offline_ok=False)
cfg = EXPERIMENTS["exp_3"]
runner = DeploymentRunner(cfg, catalog=CATALOG)
state = runner.launch()
# ... use state.base_url / state.api_key ...
runner.terminate()
```
